# light dark salt sugar vinegar 五合一主题

* Light颜色（基础配色）
* dark颜色（基础配色）
* Salt护眼颜色（在明亮模式下使用）
* Sugar护眼颜色（笔记根目录箭头彩色，标题颜色与大纲对应，在明亮模式下使用）
* Vinegar（深黑主题颜色，在暗黑模式下使用）
